# SAP Knowledge Extraction Plan

## Ziel

Verifiziertes Wissen über das SAP-System als MCP-Ressourcen bereitstellen. Dieses Wissen umfasst:

1. **Tabellen** mit Metadaten:
   - Tabellenname
   - Beschreibung
   - Spalten mit: Feldname, Datentyp, Länge, Dezimalstellen, Kurztext, Schlüsselfeld-Flag

2. **Transaktionscodes** mit Metadaten:
   - Transaktionscode
   - Titel/Beschreibung
   - Programm, Dynpro
   - Paket
   - Eingabefelder (falls verfügbar)

---

## Extraktionsmethoden

### Tabellen-Metadaten

**Quelle**: DD03L (ABAP Dictionary) via SE16 oder ABAP Report

**Schnellster Ansatz**:
```abap
REPORT zextract_fields.
SELECT tabname fieldname position keyflag datatype leng decimals
  FROM dd03l
  WHERE tabname LIKE '<PATTERN>%'
    AND as4local = 'A'
  ORDER BY tabname position.
```

Dann ALV-Grid mit `sap_read_table` auslesen.

**Für Beschreibungen**: DD04T (Datenelement-Texte) joinen:
```abap
LEFT OUTER JOIN dd04t ON dd04t~rollname = dd03l~rollname AND dd04t~ddlanguage = 'D'
```

### Transaktionscodes

**Quelle**: SE93 (Transaktionspflege) oder TSTC/TSTCT Tabellen
